# cendova.github.io — Website der Cendova-Suite

Statische Website (https://cendova.de/) — Landing-Page für **CendovaPlan**,
das browser-basierte Planungstool für Hüft- und Knie-Endoprothetik.

- Kein Build-Schritt: reines HTML/CSS, Fonts und Bilder werden lokal
  ausgeliefert (keine externen Dienste, keine Cookies, kein Tracking).
- Deploy: `.github/workflows/pages.yml` veröffentlicht `main` via
  GitHub Pages. Als **Org-Site** liegt sie am Domain-Root — die Demo
  (Projekt-Site `cendova-plan`) erscheint automatisch unter `/cendova-plan/`.
- Design-Quelle: Projekt „Cendova Design" (claude.ai/design),
  `marketing/landing-page.html` + Foundations.
- Screenshots: frei lizenzierte Lehr-Röntgenbilder (Hellerhoff, CC BY-SA,
  Wikimedia Commons) — Nachweis auf der Seite; keine Patientendaten,
  keine Hersteller-Schablonen.
