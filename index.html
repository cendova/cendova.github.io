04 · Installation

So funktioniert es

Von null zur fertigen Planung in vier Schritten.

Installer für Windows macOS‑ZIP (offline/USB) Code auf GitHub

macOS — Installation über das Terminal (empfohlen):

curl -fsSL https://cendova.de/install-mac.sh | bash

Terminal öffnen (⌘ + Leertaste → „Terminal"), Zeile einfügen, Enter — installiert bzw. aktualisiert CendovaPlan ohne Gatekeeper‑Blockade.

Windows: ZIP entpacken, Doppelklick auf „Installieren.cmd", den SmartScreen‑Hinweis mit „Trotzdem ausführen" bestätigen.

macOS: empfohlen ist der Terminal‑Einzeiler oben — per curl geladene Dateien tragen kein Quarantäne‑Attribut, daher greift die Gatekeeper‑Blockade unsignierter Downloads nicht. Das macOS‑ZIP (z. B. für den USB‑Stick) startet man ebenfalls über das Terminal: bash install-mac.command im entpackten Ordner. Ein Doppelklick zeigt dagegen „… nicht geöffnet", weil der Installer nicht mit einem Apple‑Zertifikat signiert ist.

Der Installer richtet Git und Node.js ein, lädt CendovaPlan und legt eine Desktop‑Verknüpfung an, die beim Start automatisch aktualisiert.
